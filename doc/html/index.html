Synopsis

This project uses the leapfrog algorithm to numerically integrate the motion of non-chaotic pendulums. It first solves the motion of a linear, damped, driven pendulum and plots graphs of position, phase space plots, and a plot of amplitude vs. frequency. It then plots the motion of a nonlinear, non-damped, non-driven pendulum, which also does not exhibit chaos.

This project can be found on Github.

More complete documentation can be found at (project_root)/doc/html/index.html.

Usage

To use, unpack tar.gz file to desired path Ensure all dependencies are installed and part of linker path In a terminal run

1 $ cd path_to_project/"C++ Simple Ideal Pendulum"
2 $ make
3 $ ./dist/Debug/GNU-Linux/c___simple_ideal_pendulum

Twenty graphs should spam the screen with science

Dependencies

This project uses:

  • Boost libraries which may be installed by
    1 brew install boost
    or
    1 sudo apt-get install libboost-all-dev
    on Mac and Linux
  • Gnuplot plotting software which can be installed by
    1 brew install gnuplot
    or
    1 sudo apt-get install gnuplot
    on Mac and Linux.
  • Gnuplot iostream which is included in the project source. It can also be installed by
    1 sudo apt-get install libgnuplot-iostream-dev
    on Linux or by using gnuplot-iostream.h on Github.
